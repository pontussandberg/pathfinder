The position of each square in the grid is numbered from left to right 0 - 2499.
Example:
row 1: 0, 1, 2, 3, 4, 5...., 49
row 2: 50, 51, 52....

Choose start and end position. Program will try to find a path from start to targeted position.

Click on any square-cell to toggle it into "wall". The program will find its way around it if its possible.
